# Telegram Group Manager Bot

یک ربات ساده برای مدیریت گروه تلگرام:
- خوشامدگویی به اعضای جدید
- حذف پیام‌های دارای لینک
- دستور `/ban` برای بن کردن کاربران

## نصب و اجرا
```bash
git clone https://github.com/USERNAME/telegram-group-bot.git
cd telegram-group-bot
pip install -r requirements.txt
python bot.py
